Numar aleator!

Declarati o variabila N in care sa puneti un numar aleator de la 0 la 10. Cereti utilizatorului sa introduca un numar si verificati daca este cel din variabila N. Daca nu este, afisati "Not matched! Please try again!" si mai cereti inca o valoare. In cazul in care userul a ghicit numarul afisati "Good work!".

  • Modificati scriptul astfel incat sa afisati utilizatorului si numarul de incercari din care a ghicit valoarea aleatoare.
  • Modificati scriptul astfel incat numarul aleator N sa fie intre 0 si 20, iar pentru fiecare raspuns al utilizatorului sa afisati:
    • "rece" daca numarul utilizatorului nu este in intervalul [N - 3, N + 3]
    • "caldut" daca numarul utilizatorului este in intervalul [N - 3, N + 3]
    • "cald" daca numarul utilizatorului este in intervalul [N - 2, N + 2]
    • "fierbinte" daca numarul utilizatorului este in intervalul [N - 1, N + 1]
